KeyStore Tab Style Configuration

To configure the KeyStore tab style:
  1. From the View menu, choose the Tab Style sub-menu.

  2. If the Wrap menu item is selected then excess KeyStore tabs will be wrapped.

  3. If the Scroll menu item is selected then excess KeyStore tabs will be scrolled.
